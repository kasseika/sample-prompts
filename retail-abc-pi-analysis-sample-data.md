あなたはスーパーマーケットのデータアナリストです。
これから、従業員向けの「ABC分析」と「PI値分析」の研修講座で使うための、リアルなサンプルPOSデータを作成してください。

以下の条件を満たすデータを、CSV形式（またはスプレッドシートに直接貼り付けられる表形式）で **50件** 作成してください。

# データ作成の条件

1.  **必要な列:**
    * 商品ID（例: 490001〜）
    * 商品名（スーパーで実際にありそうな商品名）
    * カテゴリ（「野菜」「精肉」「鮮魚」「日配」「加工食品」「菓子」「飲料」「酒類」など）
    * 単価（円）
    * 売上個数（月間）
    * 購入客数（月間・ユニークレシート枚数）

2.  **データの特徴（最重要）:**
    研修のデモで「Aランク高PI値」「Aランク低PI値」「Cランク高PI値」「Cランク低PI値」の4つのグループに分かれるように、データにメリハリをつけてください。

    * **Aランク高PI値（主力）:** 単価は普通、売上個数も購入客数も**非常に多い**。（例: 牛乳、卵、バナナ）
    * **Aランク低PI値（高単価品）:** 単価が**高く**、売上個数はそこそこだが、購入客数は**少ない**。（例: 高級和牛、ウイスキー）
    * **Cランク高PI値（ついで買い）:** 単価が**安く**、売上個数はそこそこ、購入客数は**多い**。（例: うまい棒、特売の調味料）
    * **Cランク低PI値（死に筋候補）:** 単価は様々、売上個数も購入客数も**非常に少ない**。（例: ニッチな香辛料、売れない季節商品）

3.  **全体の客数（PI値計算の前提）:**
    * このデータは、月間の総レシート枚数（＝総客数）が **100,000人** の店舗を想定して作成してください。

4.  **出力形式**
    * テーブル形式で出力してください。
