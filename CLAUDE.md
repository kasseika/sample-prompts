# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリの概要

このリポジトリは、LLMプロンプトのサンプルを保存・管理するためのドキュメントリポジトリです。コードベースやビルドプロセスはなく、Markdownファイルでプロンプトテンプレートを管理します。

## ファイル構成

- `README.md` - リポジトリの説明
- `*.md` - 各種プロンプトのサンプルファイル

## 作業時の注意点

- 新しいプロンプトサンプルを追加する際は、わかりやすいファイル名を使用する
- 日本語でプロンプトを記述する場合が多い
- プロンプトの目的や使用方法をファイル内に明記する
